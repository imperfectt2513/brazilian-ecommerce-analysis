# Brazilian eCommerce Analysis 🛒📊

End-to-end data analysis project on 100K+ real Brazilian eCommerce orders.

## 🎯 Project Objective
Analyze customer behavior, revenue trends, and geographic patterns 
to generate actionable business insights.

## 🛠️ Tools Used
- **SQL Server** — Data extraction, transformation, window functions, CTEs
- **Excel** — Interactive dashboard, pivot tables, slicers, KPI reporting

## 📂 Project Structure
| File | Description |
|---|---|
| `ecommerce_analysis.sql` | All 9 SQL queries with comments |
| `01 to 09 CSV files` | Query outputs used for dashboard |
| `ecommerce_dashboard.xlsx` | Interactive Excel dashboard |
| `Business_Insights.docx` | Business recommendations per query |

## 📊 Key Findings
- **São Paulo** contributes ~15% of total revenue alone
- **bed_bath_table** is the top revenue category at 8.37% share
- **99,441 total orders** across 2016–2018
- High value customers exist in **every state**, not just major cities
- Order volume grew **20x** from late 2016 to peak in late 2017

## 🔍 SQL Concepts Covered
- JOINs and aggregations
- Window Functions — DENSE_RANK, ROW_NUMBER, Running Totals
- CTEs and nested CTEs
- Correlated Subqueries
- NULL handling and data cleaning

## 📈 Dashboard Features
- 4 interactive pivot charts
- 3 slicers for dynamic filtering
- KPI Summary sheet
- City, category and time-based analysis
